Carousel Snap
=============

A jQuery plugin that creates a percentage circle based on pure CSS.

### Demo



### How to use

1. Prepare the markup holder

	```HTML
	<div id="carousel"></div>
	```

2. Invoke the carouselSnap() function

	```JavaScript
	$( '#carousel' ).carouselSnap();
	```
